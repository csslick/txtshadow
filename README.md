# txtShadow
Dynamic Text Shadow jQuery Plug-In
==================================================

##Usage
```
$(function(){
  // Use like this
  $('.shadow').txtShadow();
});
```

##Options
```
$('.shadow').txtShadow({
  blur: "10px", // text blur value
  color: "red"  // shodow color value
});
```
